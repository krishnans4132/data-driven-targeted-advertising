# A Business Analytics Study of Advertisement Fatigue in Mobile Applications

## 📌 Project Overview
This project analyzes user experience and advertisement fatigue in mobile applications using primary survey data.

The objective is to identify patterns in ad exposure, user frustration, uninstall behavior, and willingness to pay for ad-free versions.

## 🎯 Objectives
- Measure advertisement fatigue using structured survey data
- Identify which app categories generate the most ad frustration
- Analyze uninstall behavior due to advertisements
- Evaluate user tolerance toward different ad formats
- Provide business recommendations for ad optimization

## 📊 Data Collection
- Primary data collected through Google Forms survey
- Target responses: 1000
- Structured, anonymous responses
- Mostly closed-ended questions for quantitative analysis

## 🛠 Tools Used
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook
- Excel (initial cleaning)

## 📈 Analysis Performed
- Descriptive statistics
- Cross-tabulation
- Advertisement fatigue index computation
- Segment-based comparison (age, usage category)
- Behavioral impact analysis

## 💡 Key Insights (To be updated after analysis)
- Highest ad fatigue observed in [category]
- X% users reported uninstalling apps due to ads
- Rewarded ads are most tolerated format
- Willingness to pay increases with screen time

## ⚠️ Ethical Considerations
- No personal identifying information collected
- Data analyzed in aggregated form
- Used strictly for academic purposes
